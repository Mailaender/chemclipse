Controls

Chromatogram Editor

← and → When zoomed (lasso drag) use arrow keys or the scroll bar to move through the chromatogram.
Mouse Wheel Change the intensity axis which results in a vertical zoom.
Ctrl + ← or → Select the previous/next scan.
Ctrl + Alt + Left Mouse Click Select a peak.
Shift/Alt + Left Mouse Click Zoom in starting from top left.
Shift + Left Mouse Double Click Reset zoom (1:1)
